terminalにて、
	git init			「.git」フォルダーが作られる。ステージングエリアやローカルリポジトリが作成
	git add .			ワークツリーからステージングエリアへとファイルを上げます
	git commit -m "first commit"	 -m "message" とすることでエディタを起動せずにメッセージを残す
